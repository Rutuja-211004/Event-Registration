📌 Event Registration System (Flask + Bootstrap)

A simple web-based Event Registration System built using Flask, HTML, CSS, and Bootstrap.
This application allows an admin to log in and manage participants for different events.

🚀 Features
🔐 Admin Login System
📝 Register participants for events
📊 Dashboard to view all participants
✏️ Edit participant details
❌ Delete participants
🎨 Responsive UI using Bootstrap
🎯 Events Available

Coding Hackathon

🛠️ Tech Stack
Backend: Python (Flask)
Frontend: HTML, CSS, Bootstrap
Templating Engine: Jinja2
